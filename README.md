# C++ Function Problem Solving

This repository contains a collection of small to medium problem-solving exercises solved using **C++ functions**.

The main goal of this repository is to practice:
- Problem solving skills
- Using functions effectively
- Writing clean and organized C++ code

## Repository Content
- Around 50 problem-solving exercises
- Each problem is solved using at least one function
- Simple and readable code structure
- One problem per file

## Topics Covered
- Basics
- Conditions
- Loops
- Arrays
- Strings
- Math problems
- Recursion

## Why Functions?
Using functions helps to:
- Break problems into smaller parts
- Improve code readability
- Make code reusable
- Enhance logical thinking

## Language
- C++

## Purpose
This repository is created for learning and practice purposes.
